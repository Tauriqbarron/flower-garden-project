# auckland.garden — Brand Identity Notes

## Source Image
- `launch-hero.png` — LinkedIn launch hero, April 2026
- Used as the foundation for brand identity exploration

## Color Palette (extracted from launch image)

| Name             | Hex       | Usage                          |
|------------------|-----------|--------------------------------|
| Forest Green     | `#1B4332` | Primary — backgrounds, headers |
| Terracotta       | `#C65D3B` | Accent — dot separator, CTAs   |
| Cream White      | `#FFF8F0` | Text on dark, light backgrounds |
| Sage Green       | `#A7C4A0` | Secondary — borders, subtle UI |
| Pohutukawa Red   | `#C41E3A` | Accent — highlights, alerts    |
| Gold (subtle)    | `#D4A843` | Geometric accents, premium feel |

## Typography Direction
- Clean, modern sans-serif (Inter / similar)
- Bold weight for hero/display text
- Terracotta dot (`.s`) as brand signature element

## Visual Identity Direction
- **Style:** Hybrid — organic botanical illustration meets clean modern tech
- **Botanicals:** NZ native plants (pohutukawa, nikau palm, ponga fern) + garden flowers (ranunculus, dahlias, sweet peas) + herbs/seedlings
- **Tech layer:** Subtle geometric line accents (gold/cream), clean layouts
- **Density:** Lush, full-bleed, no wasted space
- **Mood:** Warm, inviting, premium, nature × technology

## Brand Signature
- The `.` in `auckland.garden` is a deliberate brand element — terracotta orange, always distinct
- Domain IS the brand name — `.garden` TLD is part of the identity
